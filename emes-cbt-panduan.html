Panduan Pengguna Emes CBT

Dokumentasi lengkap instalasi, konfigurasi, dan penggunaan seluruh fitur Emes CBT untuk administrator, guru, dan pengawas.

Versi 7.0 © Emes EduTech Google Apps Script

Pengantar

Emes CBT adalah platform Computer-Based Test (ujian berbasis komputer) yang berjalan sepenuhnya di atas Google Sheets dan Google Apps Script. Tidak memerlukan server hosting, domain, atau database eksternal — cukup satu akun Google.

Sistem ini dirancang khusus untuk kebutuhan ujian di sekolah Indonesia, dengan antarmuka berbahasa Indonesia, fitur anti-kecurangan, dan kemampuan cetak dokumen resmi berlisensi.

Arsitektur Sistem

Database → Google Sheets
Semua data (siswa, soal, nilai, log) disimpan di sheet dalam spreadsheet Google milik Anda. Data 100% ada di akun Google Anda sendiri.
Backend → Google Apps Script (Library)
Logika aplikasi berjalan sebagai Library Apps Script. Kode terlindungi — pengguna tidak perlu menyalin atau memodifikasi kode apapun.
Frontend → Web App dari Apps Script
Antarmuka disajikan melalui URL Web App yang dibuat saat deploy. URL ini diakses oleh admin maupun siswa melalui browser biasa.
Gratis sepenuhnya. Tidak ada biaya berlangganan, biaya server, atau biaya hosting. Emes CBT berjalan dalam kuota gratis Google Apps Script.

Instalasi & Setup

Proses instalasi dilakukan sekali saja. Ikuti langkah-langkah berikut secara berurutan.

1
Salin Template Spreadsheet
Klik link berikut untuk menyalin template ke Google Drive Anda:

Salin Template ke Google Drive

Google akan meminta konfirmasi. Klik "Buat salinan". Spreadsheet akan muncul di Google Drive Anda dengan nama "Salinan dari Emes CBT Template". Anda bisa mengganti namanya.
2
Buka Apps Script Editor
Di spreadsheet yang sudah disalin, klik menu Ekstensi → Apps Script. Editor Apps Script akan terbuka di tab baru.
3
Pastikan Kode di Code.gs Sudah Benar
File Code.gs di editor harus berisi tepat dua baris berikut. Jika template sudah benar, kode ini sudah ada.
Code.gs
function doGet(e) { return EmesCBTApp.doGet(e); } function doPost(e) { return EmesCBTApp.doPost(e); }
4
Jalankan Setup Awal (Sekali Saja)
Di editor Apps Script, tambahkan fungsi berikut sementara ke Code.gs, lalu pilih fungsi runSetup dari dropdown dan klik Jalankan:
Tambahkan sementara di Code.gs
function runSetup() { EmesCBTApp.setupSpreadsheet(); }
Izinkan akses yang diminta Google (akses spreadsheet). Setelah berjalan, akan muncul dialog konfirmasi bahwa setup berhasil dan semua sheet telah dibuat. Hapus fungsi runSetup setelah selesai.
5
Deploy sebagai Web App
Di editor Apps Script, klik Deploy → Deployment baru.
Isi konfigurasi:
PengaturanNilai
Tipe deploymentWeb App
Jalankan sebagaiSaya (akun Google Anda)
Siapa yang punya aksesSemua orang
Klik Deploy. Salin URL Web App yang diberikan — ini adalah alamat aplikasi Emes CBT Anda.
6
Login Pertama & Ubah Password
Buka URL Web App di browser. Login dengan kredensial default:
FieldNilai Default
Usernameadmin
Passwordadmin123
Segera ubah password default! Masuk ke menu Sistem → Pengaturan setelah login pertama.
Update aplikasi: Jika ada versi baru Emes CBT, Anda cukup melakukan Deploy ulang (Deployment baru) dari editor Apps Script. Data tidak akan terhapus.

Login & Akses

Halaman login Emes CBT menampilkan dua tab: Login Siswa dan Login Admin. Keduanya menggunakan URL yang sama.

Login Admin / Guru / Pengawas

Gunakan tab "Login Admin". Masukkan username dan password yang telah didaftarkan, lalu selesaikan CAPTCHA visual yang muncul. Sistem akan mengarahkan ke dashboard sesuai peran.

Login Siswa

Siswa menggunakan tab "Login Siswa" dengan username dan password yang telah diberikan admin. Setelah login, siswa langsung diarahkan ke dashboard ujian mereka.

Blokir Login Ganda: Jika fitur ini aktif (default), satu akun siswa hanya bisa login dari satu perangkat pada satu waktu. Login dari perangkat lain akan memblokir sesi sebelumnya.

Token Ujian

Token ujian adalah kode verifikasi yang harus dimasukkan siswa sebelum dapat memulai ujian. Fitur ini mencegah siswa mengakses ujian di luar waktu yang ditentukan pengawas.

Cara Menggunakan Token

1
Set Token di Paket Ujian
Di form Paket Ujian, isi kolom Token Ujian dengan kode yang Anda inginkan (contoh: UJIAN-2025). Atau klik ikon kunci di daftar paket untuk kelola token.
2
Bagikan Token ke Siswa saat Ujian
Saat sesi ujian dimulai, pengawas (Proktor) mengumumkan token secara lisan di ruang ujian atau menuliskannya di papan tulis.
3
Siswa Memasukkan Token
Di dashboard siswa, paket yang memiliki token akan menampilkan kolom input token. Siswa tidak bisa memulai ujian tanpa memasukkan token yang benar.
Token tidak berlaku setelah siswa memulai ujian — siswa yang sudah mulai tidak perlu memasukkan token lagi jika melanjutkan sesi. Hapus atau kosongkan token untuk menonaktifkan fitur ini.

Panduan untuk Siswa

Ini adalah panduan yang bisa dibagikan ke siswa sebelum ujian dilaksanakan.

1
Buka URL Ujian di Browser
Buka URL aplikasi Emes CBT di browser (Chrome, Firefox, atau Edge terbaru). Gunakan tab Login Siswa. Masukkan username dan password yang diberikan guru/admin.
2
Pilih Paket Ujian
Setelah login, daftar paket ujian yang tersedia akan muncul. Pilih paket yang sesuai dengan instruksi pengawas. Jika ada token, masukkan kode yang diberikan.
3
Kerjakan Soal dengan Teliti
Pilih jawaban untuk setiap soal. Jawaban tersimpan otomatis setiap beberapa detik — tidak perlu menekan tombol simpan manual. Gunakan tombol navigasi untuk berpindah antar soal. Soal yang belum dijawab akan ditandai berbeda di panel navigasi.
4
Submit Ujian
Setelah semua soal selesai (atau waktu habis), klik tombol Selesai & Submit. Ujian akan otomatis tersubmit saat waktu habis. Setelah submit, jawaban tidak bisa diubah.
5
Lihat Hasil (Jika Diizinkan)
Jika admin mengaktifkan tampilan nilai, hasil ujian akan langsung muncul setelah submit. Jika tidak, siswa melihat pesan konfirmasi bahwa ujian telah berhasil disubmit.
Jangan tutup browser atau refresh halaman saat ujian berlangsung kecuali terpaksa. Jika terjadi, login kembali dan ujian bisa dilanjutkan (jika fitur lanjut ujian aktif). Pastikan koneksi internet stabil sepanjang ujian.

Koreksi Uraian

Menu Hasil & Analisa → Koreksi Uraian. Khusus untuk paket yang memiliki soal tipe Uraian, guru harus mengoreksi jawaban secara manual di sini.

1
Pilih Paket yang Mengandung Soal Uraian
Dari dropdown di bagian atas, pilih paket yang ingin dikoreksi. Hanya paket dengan soal uraian yang akan muncul dalam daftar.
2
Klik Nama Siswa untuk Buka Detail
Daftar siswa yang sudah mengerjakan akan muncul. Klik nama siswa untuk membuka panel koreksi. Jawaban siswa ditampilkan di samping kolom input skor.
3
Beri Skor dan Simpan
Masukkan skor untuk setiap soal uraian (0 hingga bobot soal tersebut). Klik Simpan Skor Uraian. Nilai akhir siswa akan diperbarui otomatis setelah semua soal uraian dikoreksi.

FAQ Chatbot Siswa

Menu Dokumen & Utilitas → FAQ Chatbot. Siswa bisa bertanya seputar ujian melalui ikon chat di pojok bawah halaman ujian. Admin mengelola basis pengetahuan (FAQ) yang menjadi sumber jawaban chatbot.

Menambah FAQ

Klik Tambah FAQ, isi Pertanyaan dan Jawaban, lalu simpan. Pertanyaan yang mirip dengan yang ditanyakan siswa akan dicocokkan secara otomatis (pencarian teks).

Beberapa FAQ default sudah disertakan saat setup, antara lain cara memulai ujian, auto-save, waktu habis, dan browser tertutup. Anda bisa mengedit atau menghapus FAQ default ini.

Backup & Reset Database

Menu Sistem → Pengaturan, panel kanan Operasi Data. Tersedia untuk Administrator saja.

Backup Data

Klik Backup Sekarang. Sistem akan menyalin seluruh spreadsheet (termasuk semua sheet) ke folder Google Drive Anda dengan nama dan timestamp. Proses ini beberapa detik hingga satu menit tergantung ukuran data.

Disarankan membuat backup sebelum mengadakan ujian besar dan secara berkala setiap minggu. Link file backup langsung diberikan setelah proses selesai.

Reset Database Ujian

Menghapus seluruh data jawaban, nilai, dan log siswa. Data siswa, soal, dan paket tetap tersimpan.

PERINGATAN: Tindakan ini tidak dapat dibatalkan! Selalu buat backup sebelum melakukan reset. Untuk konfirmasi, sistem meminta Anda mengetik RESET_SEMUA_DATA sebelum proses berjalan.

Lisensi NPSN

Beberapa fitur premium Emes CBT (cetak dokumen resmi, export analisa) memerlukan NPSN yang terdaftar. Pendaftaran NPSN dilakukan melalui pengembang.

Cara Mendaftarkan NPSN

1
Isi Identitas Sekolah di Pengaturan
Masuk ke Sistem → Pengaturan. Isi semua kolom identitas sekolah: Nama Sekolah, NPSN, Nama Kepsek, Kabupaten/Kota, Alamat. Klik Simpan Pengaturan.
2
Klik "Daftarkan Sekolah" jika NPSN Belum Terdaftar
Jika setelah cek lisensi muncul status "Tidak Valid", tombol Daftarkan Sekolah akan tampil di samping tombol Simpan. Klik tombol tersebut untuk membuka WhatsApp pengembang dengan data sekolah terisi otomatis.
3
Konfirmasi via WhatsApp
Kirim pesan ke pengembang. Setelah NPSN diverifikasi dan didaftarkan, kembali ke Pengaturan dan klik Cek Lisensi untuk memverifikasi status berlisensi.
