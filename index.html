Cara Kerja

Setup dalam
hitungan menit

Tidak perlu hosting, tidak perlu domain, tidak perlu database. Cukup Google Spreadsheet Anda.

Apps Script Console
› Spreadsheet template dibuka
✓ Sheet siswa, soal, paket siap
› Deploy sebagai Web App
✓ URL web app dibuat
› Setup awal dijalankan
✓ Admin pertama terdaftar
› Emes CBT siap digunakan
✓ Login & mulai ujian
1
Salin Template Spreadsheet
Copy template Google Sheets yang sudah disiapkan ke akun Google Anda. Semua sheet dan struktur data sudah ada.
Salin Aplikasi ke Google Drive
2
Pasang Library & Deploy
Tambahkan Library EmesCBTApp ke Apps Script, lalu deploy spreadsheet sebagai Web App. Cukup dua baris kode.
3
Jalankan Setup Awal
Buka URL web app, jalankan setup otomatis untuk membuat akun admin pertama dan konfigurasi dasar sistem.
4
Input Data & Mulai Ujian
Import data siswa via CSV, buat bank soal, susun paket ujian, bagikan link ke siswa. Ujian bisa langsung dimulai.
Lebih Dalam

Fitur yang
benar-benar lengkap

Didesain untuk kebutuhan ujian nyata di sekolah Indonesia.

Setiap siswa mendapat urutan soal dan pilihan jawaban yang berbeda secara acak. Meminimalkan mencontek tanpa perlu buat soal berbeda.
Aktifkan token untuk paket ujian tertentu. Siswa hanya bisa masuk ujian setelah memasukkan token yang dibagikan pengawas di ruang ujian.
Jawaban siswa tersimpan otomatis. Jika browser tutup atau koneksi putus, siswa dapat melanjutkan dari soal terakhir (dapat dikonfigurasi admin).
Salin seluruh spreadsheet ke Google Drive dengan satu klik sebagai cadangan data. Tidak ada data yang hilang.
Setiap aksi admin dan siswa tercatat. Lacak siapa mengubah apa dan kapan untuk keperluan audit dan keamanan.
